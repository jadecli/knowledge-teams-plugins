# /research

Conducts a structured deep research task, synthesising multiple sources
into a coherent report or brief.

## Depth levels

| Level | Description | Est. token budget |
|-------|-------------|-------------------|
| shallow | Quick overview, 3–5 sources | ~10K tokens |
| deep | Comprehensive analysis, 10+ sources | ~50K tokens |
