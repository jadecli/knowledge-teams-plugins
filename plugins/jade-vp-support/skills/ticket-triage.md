# Ticket Triage Skill

## Activation

Auto-fires when the user pastes a customer message, support ticket, or
bug report.

## Triage framework

1. **Classify** — bug, question, feature request, or complaint
2. **Prioritise** — P1/P2/P3/P4 based on impact and urgency
3. **Route** — assign to correct VP domain
4. **Draft response** — empathetic, direct, actionable

## Response template

```
Hi {name},

Thank you for reaching out. [Acknowledge the issue in one sentence.]

[Diagnosis or answer — be specific, no jargon.]

[Next steps or resolution timeline.]

[If P1/P2: escalation path and ETA.]

Best,
{Agent name} | Jade Support
```

## Escalation triggers

- P1: Page on-call immediately, CEO notification
- Data loss: Freeze writes, notify legal
- Security incident: Hand off to jade-vp-security
