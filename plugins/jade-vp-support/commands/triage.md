# /triage

Analyses a support ticket, assigns priority, and drafts a customer response.

## Priority guide

| Priority | Criteria |
|----------|---------|
| P1 | System down, data loss risk, security incident |
| P2 | Major feature broken, no workaround |
| P3 | Feature degraded, workaround available |
| P4 | Question, enhancement request |
