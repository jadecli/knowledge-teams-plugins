# /search

Performs a cross-tool federated search across connected knowledge sources
and returns ranked, deduplicated results.

## Sources

- `code` — Codebase (via GitHub MCP)
- `docs` — Documentation files
- `tasks` — Open tasks and issues
- `all` — All connected sources (default)
