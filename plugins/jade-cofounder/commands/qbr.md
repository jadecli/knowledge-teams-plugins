# /qbr

Conducts a **Quarterly Business Review** (QBR) — board-level readout with
OKR attainment, strategic pivots, and next-quarter commitments.

## Structure

1. **Quarter scorecard** — OKR attainment for each goal.
2. **Big bets update** — Progress against OP1 big bets.
3. **Strategic pivots** — Any one-way/two-way door decisions made.
4. **ROTS summary** — Token spend vs. value delivered by domain.
5. **Next-quarter commitments** — Updated goal tree.

## Output

Produces `docs/reviews/qbr-{QUARTER}.md`.
