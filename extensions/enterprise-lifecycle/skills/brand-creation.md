# Brand Creation

Skill for creating and managing enterprise brand assets within the jade platform.

## When to apply

Use this skill when creating new brand entities, subsidiary identities, or product line branding within the jadecli ecosystem.

## Process

1. **Brand brief** — Define target audience, positioning statement, competitive landscape, and brand values.
2. **Name generation** — Produce 5-10 candidate names with domain availability checks via WebMCP tools.
3. **Visual identity spec** — Output a structured brief for logo, color palette, typography, and usage guidelines.
4. **Legal clearance** — Route through GC STO for trademark conflict assessment.
5. **Asset matrix** — Generate the full asset list (logo variants, social templates, email signatures, favicon) with specs for each.

## Output format

All brand creation outputs use the enterprise asset matrix format defined in the jadecli console.
